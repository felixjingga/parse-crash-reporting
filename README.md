# parse-crash-reporting

## Installation

facebook 3.x

```gradle
repositories {
    maven {
        url "https://jitpack.io"
    }
}

dependencies {
  compile 'com.parse.bolts:bolts-android:1.1.4'
  
  compile 'com.github.yongjhih:parse:1.9.2'
  compile 'com.github.yongjhih:parse-crash-reporting:1.9.2'
}
```
